# Chime TTS Say Service

There are 4 services added by Chime TTS:

1. The [`chime_tts.say`](./say-service) service generates a local MP3 file comprised of chime/s and/or TTS audio, for playback.

2. The [`chime_tts.say_url`](./say_url-service) service returns local/remote filepaths to files generated by [`chime_tts.say`](./say-service), without playback.

3. The [`chime_tts.replay`](./replay-service) service repeats the last service call to [`chime_tts.say`](./say-service) with the previous parameter values.

4. The [`chime_tts.clear_cache`](./clear_cache-service) service selectively removes cached audio files.
