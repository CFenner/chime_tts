# `chime_tts.replay` Service

The `chime_tts.replay` service calls the [`chime_tts.say`](./say-service) service with the exact same set of parameter values used in the previous [`chime_tts.say`](./say-service) service call.

## Parameters
There are no parameters to set for this service.

## YAML Example

```
service: chime_tts.replay
data:
```
