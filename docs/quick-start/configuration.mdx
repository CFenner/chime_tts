# 3. Configuration

You can customise the following elements for the `chime_tts.say` and `chime_tts.say_url` services through the configuration page, located at:

`Settings` --> `Devices & services` --> `Chime TTS` --> `CONFIGURE`

<img src={useBaseUrl('/img/quick-start/configuration.png')} />


| Config Option | Description | Default Value |
|-------------------------|----------------------------------------------------------------------------------------------------------------------------|--------------------------------------------------|
| **Timeout** | Set the time (in seconds) to wait before service calls are terminated. | `60` |
| **Default TTS Platform** | The TTS platform to use if service calls are missing the `tts_platform` parameter. | *The first TTS platform configured in your Home Assistant instance.* |
| **Default Offset** | The default offset (in milliseconds) to insert between chimes & TTS audio. Positive values will add a delay and negative values will overlay audio segments on top of each other. | `450` |
| **Media Folder** | The name of your media folder. | `local` |
| **Downloaded Chimes Folder**| The local folder path to store downloaded chime mp3 files. | `{root}/media/sounds/temp/chime_tts/` |
| **Temporary MP3 Folder**| The local folder path to store the generated mp3 files. | `{root}/media/sounds/temp/chime_tts/` |
| **chime_tts.say_url Folder** | The folder path to store MP3 files generated by the [chime_tts.say_url](../say_url-service/service) service. | `{root}/config/www/chime_tts/`
| **Custom Chimes** | Add up to 5 file paths / URLs to your own custom chime files. | None |
